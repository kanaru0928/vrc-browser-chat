-----------------------------
 VRC Browser Chat (β) v1.2.1
-----------------------------

ダウンロードいただきありがとうございます！
このアプリはVRChatのチャットボックスをブラウザから送信できるツールです。

■ インストール方法
同梱した「vrc-browser-chat-server_1.2.0_x64_en-US.msi」を実行して、インストールを完了させてください。

■ 使用方法
1. VRChatを起動します
2. 本アプリを起動し、OSCセクションの「Start」ボタンをクリックします
3. 「Web Server」セクションに記載されたURLにアクセスします (Openボタンをクリックすると自動で開きます)
4. ブラウザからチャットメッセージを入力して送信します
5. VRChatのチャットボックスにメッセージが表示されます
